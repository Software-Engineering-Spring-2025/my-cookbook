React Documentation

CookBook is a platform to search for recipes that can be made using the available ingredients. This page provides information regarding the React components of the application.

AppContent.tsx

The component defines the routes for the application and decides which component on render on that particular route.

Home Page

Landing page for the application.

List of static pages included:

Page Description
About Page Describes the purpose of the application.
Contact Page Provides contact/support information.
RecipeInformation

List of files:

File Description
getRecipeInformation.action.ts Contains action handlers (LOAD, SUCCESS, FAIL) for getting the recipe information from the DB
getRecipeInformation.actionTypes.ts Contains predefined action types since we are using Typescript.
getRecipeInformation.reducer.ts Contains a normal function that performs state update based on API call output - SUCCESS/FAIL.
getRecipeInformation.saga.ts It is a middleware, that handles the async logic in the data flow based on the action that has been dispatched
RecipeInformation.tsx This component displays images for recipe making, the procedure to make the dish and the trivia and factual info related to it. This component is a dynamic component and is seen only when you click on a recipe from the recipe list
